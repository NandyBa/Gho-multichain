# GHO Multichain


This project levrage GHO ecosystem.

## Before execution

**Define mainnet RPC**
```bash
export MAINNET_RPC_URL=<YOUR ETHEREUM MAINNET RPC>
```

## Execution

**Step 1: Launch fork**
```bash
bash src/fork-mainnet.sh
```

**Step 2: Execute GHO contract update**
```bash
bash src/UpdateGhoMainnet.sh
```
